# CC98 WPS Table Disguise

将 CC98 论坛页面转换成接近 WPS/Excel 表格的油猴脚本，用于低调浏览。脚本样式资源已内嵌，不依赖额外远程资源。

## 安装

1. 安装 Tampermonkey 或兼容的用户脚本管理器。
2. 打开 Greasy Fork 安装链接：<https://greasyfork.org/zh-CN/scripts/585813>
3. 也可以使用 GitHub raw 链接：<https://github.com/amiongogo/cc98-wps-table-disguise/raw/main/cc98-wps-table-disguise.user.js>
4. 在用户脚本管理器中确认安装或更新。

## 功能

- 首页、新帖、关注、精选、版面帖子列表、帖子内部、回复我的、@ 我的、系统通知等常用页面表格化显示。
- 帖子内部保留翻页、回复、引用、赞、踩等日常操作入口。
- 未登录时跳转原生登录页，登录页、个人中心、私信等复杂交互页面保留原界面。
- 顶部保留常用导航入口，并根据登录状态显示登录或退出登录。
- 提供油猴菜单切换 WPS 表格伪装和图片显示。
